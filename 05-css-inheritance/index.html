CSS Inheritance

Inherited properties

Notice that there's an <em> tag wrapping around the word fox. How come the word "Fox" is colored pink? If you think about it, we haven't applied any styles to the "em" element, only to paragraphs.
Since certain property in the CSS can be inheritable, when I apply pink color to paragraph elements, this color property gets inherited to paragraph's child and grand-child viz em tag in this case.

Can all CSS properties be inherited?

No, not all CSS properties are inherited. Some properties, like ‘background-color’ and ‘border’, are not inherited by default. However, you can force these properties to be inherited by using the ‘inherit’ value.
